Stop Instance

This will gracefully stop the instance. Data on instance store volumes will be lost.

Stopping an instance will cause it to lose its public IP address (if not using an Elastic IP).

Reboot Instance

The instance will be rebooted. This is equivalent to an OS-level reboot.

Rebooting preserves the public IP address and does not change the instance or its attached volumes.

Terminate Instance

This action is permanent and cannot be undone.

Warning: Terminating an instance permanently deletes it. All data on instance store volumes will be lost. EBS volumes may also be deleted depending on the configuration.
